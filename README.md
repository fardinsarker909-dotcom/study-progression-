# Study Tracker

A web-based application built with **Flask** to help students track their study hours and manage tasks.

## 🚀 Features
* Track study sessions in real-time.
* View history of completed tasks.
* Easy-to-use web interface.

## 🛠️ Technologies Used
* **Python** (Flask Framework)
* **HTML5/CSS3**
* **JSON** (for data storage)
